# SciKG Discovery Questions
# These questions drive the schema and evaluation.
# The graph must be able to answer all of these.
# Source: project review session + David consultation

## Core questions

1. Which papers used the 21T FT-ICR MS instrument?
2. Which researchers at NHMFL published FT-ICR 
   proteomics work after 2015?
3. Which grants funded top-down proteomics research 
   at MagLab?
4. Which papers cite Anderson et al. 2016 
   (10.1021/acs.jproteome.6b00696)?
5. Which instruments were used across the ICR corpus 
   and how many papers used each?
6. Which papers used both FT-ICR MS and a secondary 
   instrument such as Orbitrap?
7. Which researchers have co-authored more than one 
   ICR paper?
8. Which papers have associated datasets in MassIVE 
   or other open repositories?
9. Which funders appear most frequently across the 
   ICR corpus?
10. Which papers used ESI as the ionization method?
11. Which papers studied dissolved organic matter 
    using FT-ICR MS?
12. What publications came from the ICR facility 
    between 2020 and 2026?

## Questions dependent on raw file integration (Phase 2)

13. Which raw files were collected on the 21T FT-ICR MS?
14. Which experiments produced data that was later 
    published in paper X?
15. Which raw files have associated MassIVE deposits?

## Questions added 2026-06-29

16. Can we identify which datasets (raw files) correspond to specific 
    publications, and what experimental parameters were used (polarity, 
    scan settings, activation types, etc.)?

17. Which collaborators or institutions most frequently co-publish 
    with MagLab ICR facility users, and are there identifiable networks 
    around particular research topics?
